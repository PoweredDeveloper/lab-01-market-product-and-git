## Roles and responsibilities

* **Back-end engineer**
  Designs, implements, and maintains server-side services and APIs, ensuring scalability, reliability, and correct business logic.

* **Mobile engineer (iOS/Android)**
  Develops and maintains mobile applications, integrates backend APIs, ensures performance, and delivers a smooth user experience on mobile devices.

* **DevOps engineer**
  Builds and maintains CI/CD pipelines, manages infrastructure, monitors systems in production, and automates deployments and scaling.

* **QA engineer**
  Designs and executes test plans, performs manual and automated testing, and ensures system quality across releases.

* **Data engineer**
  Designs and maintains data pipelines, ensures reliable data ingestion and storage, and supports analytics and reporting use cases.

## Common skills across roles

* Proficiency with version control systems (Git)
* Understanding of distributed systems and microservice architectures
* Experience with cloud platforms and containerization (Docker, Kubernetes)
* API design and integration (REST, gRPC)
* Debugging, monitoring, and logging in production systems
* Basic security awareness (authentication, authorization, data protection)
* Collaboration skills and experience working in agile development teams
